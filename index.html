Как обновлять дашборд

  1. В Битрикс24 открой список задач отдела.
  2. Сделай экспорт задач в файл .xls.
  3. Переименуй файл по шаблону tasks_YYYY-MM-DD_HH-MM.xls.
  4. Положи файл в папку data/raw/.
  5. Открой файл data/exports.json и добавь имя новой выгрузки в список files.
  6. Залей изменения через GitHub Desktop: Commit to main → Push origin.
  7. Открой ссылку GitHub Pages и нажми Обновить.

Как часто делать выгрузки

Обычный режим: 1 раз в рабочий день утром до 09:30.

Активная проектная фаза: 2 раза в день — утром и после обеда.

Перед сдачей проекта: 3 раза в день — утром, в середине дня и в конце рабочего дня.

Статический дашборд. Работает на GitHub Pages без Streamlit, Python и серверной части.
